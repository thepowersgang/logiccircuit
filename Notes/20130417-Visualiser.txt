All blocks are optional, either by omission or an empty list (for input)
- Position in "blocks" (not pixels)
- Size in blocks
- Offsets from top/left for input values (in units of 1/16 blocks)
- Offsets from top/left for output values


Note: Contiguous group links are indicated as a single input/output
